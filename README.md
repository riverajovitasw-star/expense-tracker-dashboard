## Project Overview

Expense Tracker Dashboard is a full-stack web application designed to help users manage their personal finances efficiently. The system enables users to track income, monitor expenses, analyze spending patterns, and visualize financial data through interactive dashboards and charts.

Developed as part of a Software Development Internship, this project demonstrates full-stack development skills using modern web technologies.

## Installation

### Clone Repository

git clone https://github.com/riverajovitasw-star/expense-tracker-dashboard.git

cd expense-tracker-dashboard

---

## Environment Variables

Create a `.env` file inside the backend folder:

PORT=5000

MONGO_URI=mongodb://localhost:27017/expense-tracker

JWT_SECRET=your_super_secret_jwt_key_change_this_in_production

JWT_EXPIRE=30d

NODE_ENV=development

---

## How to Run

### Step 1: Start MongoDB

Make sure MongoDB is installed and running.

Windows:

net start MongoDB

### Step 2: Install Backend Dependencies

cd backend

npm install

Run Backend:

npm run dev

Backend URL:

http://localhost:5000

### Step 3: Install Frontend Dependencies

Open another terminal:

cd frontend

npm install

Run Frontend:

npm run dev

Frontend URL:

http://localhost:5173
